Guia da linguagem markdown

Uma linguagem que oferece algumas propriedades da linguagem HTML, de forma mais simples e intuitiva.

O que é a linguagem Markdown

A linguagem markdown é uma linguagem de marcação que visa a construção de documentos de textos que possuem tabelas, listas, cabeçalhos e imagens que acompanham o conteúdo e ajudam a montar blocos de textos simples que ampliam a visão do leitor sobre determinado assunto e mostra credibilidade nos projetos que tem documentos README, que tem como principal linguagem, o Markdown. A linguagem surgiu com a necessidade de simplificar a forma de criar documentos que contem marcação, como a linguagem de marcação principal anteriormente era o HTML, sendo essa, muito complicada de se ler e compreender plenamente, a linguagem markdown nasceu com o objetivo de ser uma simplificação do HTML, para tornar as manutenções nos códigos algo menos demorado.

Foi com esse pensamento que John Gruber junto com .. criou a linguagem em 2004, desenvolvida exclusivamente para ser simples e "readable".

Utilidades no GitHub

Desde já, digo que a linguagem markdown tem um papel fundamental tanto no GitHub, quanto em redes sociais de programadores em geral, como o Stack Overflow. A linguagem é simples e intuitiva

Tanto no desenvolvimento de repositórios quanto na interação de pessoas no GitHub é feita através

Construção do README

O markdown é crucial para a estruturação do README, um arquivo, normalmente em markdown, que traz uma perspectiva geral de um projeto/repositório no momento de vistoria.

Interação em Issues e criação de textos

Utiliza-se também da tecnologia para se comunicar com usuários que usam o GitHub, para ajuda-los a resolver um problema ou solucionar algum bug nos códigos de uma certa aplicação.

Sintaxes do Markdown

Por priorizar a simplicidade e a facilidade de leitura no code, o markdown segue um padrão bem simples na composição de suas sintaxes, para ajudar os coders a ler e entender os códigos plenamente, sem ter que quebrar a cabeça lendo um simples paragrafos. Abaixo, coloco todas as tags de markdown existentes e as comparo com suas versões em HTML, com a finalidade de ajudar no entendimento da linguagem.

Cabeçalhos/Títulos

Para criar títulos hierarquicos em markdown é bem simples, basta aplicar a tralha(#) antes do texto e dar um espaço do que se deseja escrever, e assim seu título estará feito.

Níveis de Cabeçalho na linguagem Markdown
Níveis de Cabeçalho em documentos Markdown
Hierarquia Em HTML Em Markdown
Título 1 <H1>Título 1</H1> # Título 1
Título 2 <H2>Título 2</H2> ## Título 2
Título 3 <H3>Título 3</H3> ### Título 3
Título 4 <H4>Título 4</H4> #### Título 4
Título 5 <H5>Título 5</H5> ##### Título 5
Título 6 <H6>Título 6</H6> ###### Título 6

Paragrafos

A função mais comum de um documento markdown e de marcação também, são os paragrafos, pois os mesmos desenvolvem o conteúdo do que veremos em uma página que usa markdown. Para usar os paragrafos basta dar um espaço antes de escrever um conteúdo, e com apenas isso você já estará organizando os paragrafos no código.

Quebra de linha

Utilize "\" para quebrar a linha em um documento.

Ênfase de palavras

Em markdown, existem diversas formas de se destacar um texto, dentre elas temos:

Negrito

Para aplicar o destaque de negrito em seu texto, aplique dois asteriscos (**) em volta do texto, e você já terá o texto em negrito.

Italico

De forma similar a aplicar negrito em um texto, basta colocar um asterisco (*) ao redor do que desejar e assim você terá um texto em italico.

Negrito e italico

Há formas de dar ênfase nos textos de ambas as formas, utilizando três asteriscos (***) ao redor do termo desejado.

Listas

Listas Ordenadas

As listas ordenadas são usadas para descrever um número de passos ordenados e em sequência, sendo posto números em cada item da lista que se deseja criar. Para criar essa lista, bas colocar o número que quer que a lista siga e assim, você terá uma lista ordenada. 

                            
                                Lista de compras
                                1. Alho
                                2. Cebola
                                3. Arroz
                                4. Tomate
                                5. Açucar
                                ...

Listas Desordenadas

há a forma de lista desordenada que é usada pra citar tarefas, adicione um traço (-) ou um asterisco (*) antes do conteudo e de um espaço entre eles, desse jeito, a lista já estará criada. 

                            
                                - Lavar as roupas
                                   * Colocar na máquina
                                   * Colocar no varau
                                   * Passar ferro

Códigos

Como códigos são um dos principais assuntos quanto se trata de criar documentos markdown e se conectar com outras pessoas, a linguagem também tem sintaxes para descrever códigos, podendo ser demonstrado de multiplas maneiras:

Códigos em linhas

Utilize a Crase (`) ao redor de um termo e assim você criara uma exemplificação de um código de forma inline: 

                            
                                Utilize `Background-color: blue;` para mudar a cor de fundo pra azul.

Blocos de códigos

Há também a forma mais comum e útil, que serve para demonstrar códigos mais complexos e maiores que os citados anteriormente, pra isso é necessário criar caixas grandes que contem os códigos com sua formatação original, para isso, vá a uma linha em branco e circumde o código desejado entre três crases (```). 

                            
                                ```
                                div#main-header {
                                    Background-color: blue;
                                    color: white;
                                }
                                ```

Linhas Horizontais

As linhas horizontais ou Linhas de Cabeçalho tem seu papel em fazer separações semânticas entre um cabeçalho e um paragrafo, para utilizar as lihas horizontais vá a uma linha abaixo de onde deseja utiliza-la e escreva três traços (---) ou três asteriscos (***)

Links

Aplique o texto de referência entre os colchetes e o link onde se deseja colocar o link entre os parênteses, dessa forma você ira imitar a tag de link do HTML em Markdown, e conectará a um novo endereço.

HTML

Embora a linguagem markdown seja uma linguagem de marcação própria, ela também pode ser compativel com o HTML, em algumas ocasiões, como o código-fonte do README, pode se usar algumas tags que se usa em HTML normalmente, e até mesmo um pouco de CSS (em formatação inline).

Sintaxes extendidas

Em situações mais casuais, as tags simples do markdown já ajudam o bastante para criar documentos de textos simples e intuitivos, porém, existem extensões das tags que oferecem mais possibilidades para as formatações, sendo essas utilizadas em explicações especificas ou para documentar algo mais complexo, como tabelas, elementos químicos, equações matematicas entre outros.

Porém, essas tags extendidas nem sempre seram compatíveis com o lugar que você usa elas, por isso existem alguns conjuntos adicionais.

Veja a seguir as extensões das tags:

Tabelas

Basta utilizar barras verticais e hifens: 

                            
                                | Nome      | Nota    |
                                | --------- | ------- |
                                | Luciano   | 10      |

A largura das grades também podem variar, como mostrado abaixo: 

                            
                                | Nome | Nota |
                                | ---- | ---- |
                                | Leonardo | 10 |

Alinhamento em tabelas

Podemos alinhar textos nas colunas para a esquerda, para direita e no centro apenas ao colocar dois pontos (:) em uma das celulas da coluna. Para escolher o lado que se deseja alinhar, basta colocar os dois pontos na direção desejada, sendo , e nos dois lados para alinhar no centro. 

                            
                                | Sintaxe     | Uso     | Texto       |
                                | :---        | :----:  |        ---: |
                                | Cabeçalho   | Título  | *Descrição* |
                                | Paragrafo   | Texto   |             |

Códigos em bloco cercados

Porém, há a versão mais completa, os chamados blocos de código cercados, que permite cancelar indentação formatada do documento no modelo e apenas aplicar quanta indentação desejar. 

                            
                                ```

                                    "Primeiro nome": "João",
                                    "Sobrenome": "da Silva",
                                    "Idade": "46"

Notas de Rodapé

Identificadores de Cabeçalho

Assim como o HTML, o markdown permite criar identificadores em seus cabeçalhos/títulos para formatações diversas e acesso rápido a pedaços de seus textos. 

                            
                                ## Dissecando o Projeto {#diss-pro}

Em HTML: 

                            
                                <h2 id="diss-pro">Dissecando o Projeto</h3>

Ligação aos identificadores de Cabeçalhos

A função acima é dada a capacidade de se ler o conteúdo mais rapidamente, permitindo o usuário de se locomover e explorar os documentos desejados de forma mais dinâmica e intuitiva, combinando os identificadores em títulos e os links.

Palavras marcadas

É possível marcar palavras apenas ao por dois tios(~~) em seus arredores, criando um resultado como esse, e informando que o texto marcado compartilhava uma ideia errônea ou desatualizada através dos textos, fazendo isso sem descartar o conteúdo original. 

                            
                                ~~2 + 2 = 5.~~ Sabemos que 2 + 2 = 4 em qualquer contexto.

A saída do código será demonstrada da seguinte forma:

2 + 2 = 5. Sabemos que 2 + 2 = 4 em qualquer contexto.

Emojis

Existem duas maneiras de se usar emoji em aplicações markdown: Copiando de algum lugar e colando nos documentos ou utilizando códigos de formatações:

Copiando emojis

Em uma tarefa simples, basta ir até a emojipedia, copiar e colar o emoji desejado em seu documento, assim o emoji aparecerá automaticamente, sem necessidade de códigos/sintaxes adicionais.

Usando códigos de atalho para emoji

Para quem é mais hardcore e pretende seguir o jeito mais rápido de se colocar emojis em markdown, use dois pontos ao redor do nome do emoji, fazendo-o se formatar da mesma forma que seria o copiando. 

                            
                                Olá :wave:, Eu sou um grande programador.
                                Prazer em conhece-lo :smile:

Destaque

Alguns processadores de markdown permitem dar destaque no texto de uma forma alternativa, utilizando dois símbolos de igual (==) e circulando o texto desejado com os sinais. 

                            
                                Preciso destacar essas ==palavras muito importantes==.

Texto subscrito

Em química, por exemplo, existem números e letras que aparecem pequenos e abaixo dos caracteres padrões, que são formatados no markdown com dois tios rodeando o texto.

Texto Sobrescrito

Para utilizar textos sobrescritos utilize cincunflexos(^) ao redor do texto, e a formatação será concluída.

Desabilitando ligações de link automática

Existe a possibilidade de desligar essa ligação automática dos links, para ocasiões em que há , basta utilizar crases (`) para rodear o link, fazendo-o inacessível através do clique.

A saída do código será demonstrada da seguinte forma:
