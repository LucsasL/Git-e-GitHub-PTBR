O que é Git e GitHub?

Git

Principal Software para Controle de Versão (VCS).

O que é?

Git é um software de controle de versionamento que manipula as diferentes versões de seu programa e as transporta a um repositório local, onde habita seus arquivos de programação e códigos-fonte. Os commits podem ser diferenciados por "galhos" que ramificam seu projeto em multiplas versões e os controla otimizando assim a organização e facilitando sua vida com o desenvolvimento de aplicações.

A sua utilização está intrinsicamente ligada a uma questão de produtividade e ordem, e os desenvolvedores a utilizam para evitar situações inconvenientes. Uma situação que muitos programadores passam durante o desenvolvimento de seus softwares, é a separação de várias pastas compactadas em seu computador com diferentes versões de um mesmo projeto, tendo em mente, a possibilidade de seu projeto desandar por um equivoco ou por erros por parte da equipe. Com isso, as pastas são criadas para que se recupere mais tarde, caso algo que não esteja no plano do cliente surja em sua construção.

Versions Control System (VCS)

A solução se encontra no uso de softwares de VCS. Os chamados Version Control System, ajudam a otimizar as versões de seus programas usando branchs, que não só organizam parte de seus projetos mas otimizam as separações e funções de seu grupo, para desenvolvimento pleno dos sistemas em questão. O Git, como foi brevemente abordado, é um desses softwares de versionamento, que ajudam a manter ordem nos projetos e facilitam a manipulação de suas mudanças através de ramificações.

Como funciona o Git?

Diferente das pastas compactadas guardadas em um computador, para a separação de versões, o Git faz um Commit a uma singularidade de dados que são separados de acordo com o branch em que se deseja colocar. O termo commit refere-se a ação de levar dados(arquivos) a um repositório central, que junta todos os os outros commits de diversas versões já realizadas de um projeto em uma determinada parte dele, podendo ser separada da forma em que o integrante do grupo deseja.

GitHub

Diferente do Git, o GitHub não se trata de um versionamento de software, ele na verdade, é uma rede social de programadores.
