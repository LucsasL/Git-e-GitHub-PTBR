O que é GitHub?

GitHub

Diferente do Git, o GitHub não se trata de um versionamento de software, ele na verdade, é uma rede social de programadores que contém repositórios remotos.

O que é?

GitHub é uma rede social que abrange o mundo da programação, e serve também como uma plataforma para seus repositórios remotos, onde há a união dos commits de um código-fonte. Além disso, há a possibilidade de conexões com novos repositórios, contribuições em projetos de código aberto e até discussões gerais sobre problemas no decorrer dos trabalhos com outros programadores e empresas.

Você pode até mesmo fazer networking com os programadores mais insanos no mundo da tecnologia, como Linus Torvalds, Mark Zuckerberg e muitos outros.

Coisas que o GitHub pode te oferecer

Além de possibilitar uma conexão integral com os participantes e corporações da programação e no desenvolvimento de tecnologias, o GitHub te possibilita a melhorar ainda mais sua situação quando falamos de compartilhamento de projetos, dentre estas vantagens, temos:

O uso e criação ilimitado de repositórios

No GitHub é possível o uso de quantos repositórios você quiser pra por seus projetos, sendo eles públicos ou não, tendo a posse de vários colaboradores nos projetos no caso de repositórios públicos e até 3 em repositórios privados

Hospedagem de códigos-fonte de projetos Web Gratuito

Além da organização oferecida pelos repositórios remotos, temos também a capacidade de hospedar projetos web que incluem tecnologias mais simples como o HTML, CSS e JavaScript com padronização ECMAScript. E isso tudo, de forma gratuita.

Acesso a códigos-fonte de programas fodas

O GitHub fornece acesso aos códigos-fonte de diversos softwares famosos e utilizados por pessoas no dia a dia, tanto por pessoas na bolha dev, quanto fora dela. Podemos citar códigos-fonte de diversos programas que tem seu próprio repositório remoto no GitHub como o do próprio Git, Linux, Wordpress, Visual Studio Code, e muito mais.

Networking com toda a comunidade de desenvolvedores

A importância da comunicação e do compartilhamento de informações é essencial para o desenvolvimento das habilidades necessárias para criação de softwares e otimização dos códigos-fonte que os erguem, e nesse quesito, o GitHub não peca, pois foi arquitetado com o objetivo de criar conexões entre programadores.

O ambiente que a plataforma oferece contribui muito para o desenvolvimento de seus conhecimentos em tudo que tange o mundo da tecnologia, pois orienta os seus usuários a compartilharem seus códigos através de ajustes de produtividade mostrado no gráfico de quadrados verdes, que incentiva os programadores a sempre aplicar modificações nos projetos e até criar novos deles, ou até discutir sobre eles na criação de issues, que são exclusivamente criadas pra isso, pois apartir de todas essas ações em seus repositórios, sua pontuação no gráfico passa a aumentar.
